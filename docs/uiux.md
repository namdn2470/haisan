# UI/UX — Hải Sản Biển Xanh

## Design Guidelines

### Màu sắc
- **Primary Blue**: `#005bdc` — thương hiệu, buttons
- **Dark Blue**: `#0040a8` — hover, active
- **Navy**: `#061d3b` — text đậm
- **Light Blue**: `#f4fbff` — background sections
- **Border**: `#e6eef8` — cards, inputs
- **Red**: `#ff2b2b` — giá, badge bán chạy
- **Green**: `#04a755` — badge tươi ngon, stock

### Typography
- Font: Inter (hệ thống)
- Heading: 800 (extra bold)
- Body: 400 regular
- Price: 700 (bold)

### Components Pattern
- **Product Card**: ảnh + badge + tên + giá + rating + add-to-cart
- **Category Card**: ảnh bo góc + tên
- **Promo Card**: ảnh full + overlay text
- **Service Strip**: icon + text ngang
- **Bottom Nav**: mobile navigation 5 items

### Layout
- Container max-width: 1440px
- Desktop: topbar → header → nav → content → footer
- Mobile: header compact + bottom nav
- Admin: sidebar (250px) + main content

### Responsive Breakpoints
- Desktop: > 1024px
- Tablet: 768-1024px
- Mobile: < 768px

## Trang chính
1. Trang chủ: hero + service strip + categories + best sellers + promo
2. Chi tiết sản phẩm: ảnh + info + buy card + size selector + processing
3. Giỏ hàng: danh sách + tổng tiền
4. Checkout: form đặt hàng
5. Admin: sidebar + stats + tables
